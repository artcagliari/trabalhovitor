# ATER Digital — Hortaliças

Site de extensão rural com orientações práticas sobre **tomate**, **alface** e **rúcula**.

## Como abrir

Abra o arquivo `index.html` no navegador (Chrome, Firefox, Safari ou Edge).

No terminal, a partir desta pasta:

```bash
open index.html
```

Ou use um servidor local simples:

```bash
python3 -m http.server 8080
```

Depois acesse: http://localhost:8080

## Conteúdo

- **Início** — visão geral e dicas gerais
- **Tomate** — plantio, adubação, doenças, colheita e cuidados
- **Alface** — plantio, adubação, doenças, colheita e cuidados
- **Rúcula** — plantio, adubação, doenças, colheita e cuidados

O site foi pensado para produtores rurais, com fonte grande, botões amplos e textos objetivos.
